**TASY v2 (Эволюционное расширение v1)** — это автономная виртуальная сущность (тамагочи), разработанная для микроконтроллеров **ESP32-C3** с использованием асинхронного фреймворка **Embassy**. 

---

## 🧠 Ключевые особенности архитектуры

*   **🎭 Эмоциональная система** — динамический расчет параметров состояния сущности: `stress`, `loneliness`, `curiosity`, `stability` и `Mood`.
*   **💾 Умная память (Memory)** — продвинутая структура данных с поддержкой типов записей, их важности и временных меток (`timestamp`). Полностью заменяет примитивный лог сообщений (Message log).
*   **🚌 Шина событий (Event Bus)** — высокопроизводительная асинхронная шина на основе бесконфликтного кольцевого буфера (`lock-free ring buffer`) с гарантированным отсутствием динамических аллокаций (`no_alloc`).
*   **🧬 Характер и личность (Personality)** — персистентное состояние, которое сохраняется в энергонезависимую память и напрямую влияет на текущее поведение.
*   **👁️ Когнитивная задача (Cognition Task)** — полностью независимый, автономный когнитивный цикл обработки информации.
*   **💤 Режимы сна (Sleep / Wake)** — поддержка как автоматического (на основе когнитивного цикла), так и принудительного ручного перевода устройства в режим энергосбережения и обратно.
*   **📈 Примитивное подкрепление** — базовая система обучения (например, показатель привязанности `attachment` динамически растет при пробуждении из одиночества).

---

## ⚙️ Технические детали

*   **Платформа:** ESP32-C3
*   **Фреймворк:** Embassy (Async Rust)
*   **Версия схемы хранения (SAVE_VERSION):** `2`
    > ⚠️ **Важно:** Миграция базы данных автоматически сбрасывает текущее состояние тамагочи при обнаружении несовместимости схем сохранения.
